# 策略 A：静态产品展示

## 适用场景
笔记重点介绍产品界面、设计、最终效果时使用。

## 使用时机
- 笔记强调"功能列表"、"界面设计"、"最终效果"
- 展示产品的外观、UI、视觉设计
- 突出产品的设计美学和用户体验

## Prompt 模板

```
8K ultra-high-definition commercial product photography, 4:3 aspect ratio,
hyper-realistic style with extreme clarity and micro-detail visibility,
cinematic studio lighting with controlled highlights and soft shadows,
no noise, no artifacts.

Main subject: '[ProductName]' brand logo in [describe font style] with
[describe icon] positioned at top center, sharp text rendering,
professional tech brand aesthetic.

Central visual: [主要产品视觉，如：3D rendered interface/product icon/concept visualization],
[describe key features to showcase], glass-like material
with subtle transparency, metallic accent highlights in [brand accent color],
floating particles suggesting [product characteristic].

Background: [brand colors] gradient ([hex color 1] to [hex color 2]) fading from
top to bottom, soft bokeh lights with [warm/cool] glow, [atmosphere description:
luxurious/professional/modern/playful].

Surface: clean reflective base with subtle light pooling, soft separated
shadow beneath main elements, controlled edge highlights.

Text overlay: Chinese text '[产品核心功能描述]' in bold white font (48pt equivalent)
positioned at bottom center with 10% opacity dark shadow for readability.

Camera angle: straight-on hero composition with slight depth perspective,
premium editorial product shot feel, centered and balanced.
```

## 填写示例

以智能表单填写工具为例（使用占位符）：

```
8K ultra-high-definition commercial product photography, 4:3 aspect ratio,
hyper-realistic style with extreme clarity and micro-detail visibility,
cinematic studio lighting with controlled highlights and soft shadows,
no noise, no artifacts.

Main subject: '[ProductName]' brand logo - [describe actual logo: e.g., "geometric
icon with initials" / "leaf symbol" / "abstract shape"] and brand text in
[font style: e.g., "clean sans-serif" / "modern rounded" / "tech-style"] font
positioned at top center, sharp text rendering, professional tech brand aesthetic.

Central visual: 3D rendered [main product visual: e.g., "PDF document" /
"dashboard interface" / "mobile app screen"] with [key features: e.g.,
"floating smart form fields" / "data visualization charts" / "AI processing indicators"],
glass-like material with subtle transparency, metallic accent highlights in
[brand accent color with hex: e.g., "#D4AF37" gold / "#FF6B35" orange],
floating particles suggesting [product characteristic: e.g., "intelligent processing" /
"real-time analysis" / "seamless workflow"].

Background: [primary brand color] to [secondary brand color] gradient
([hex value 1] to [hex value 2]) fading from top to bottom,
soft bokeh lights with [warm/cool] [accent color] glow,
[atmosphere: e.g., "luxurious professional" / "modern tech" / "creative innovative"].

Surface: clean reflective base with subtle light pooling, soft separated
shadow beneath main elements, controlled edge highlights.

Text overlay: Chinese text '[产品核心功能的中文描述：如"智能表单填写"]'

premium editorial product shot feel, centered and balanced.
```

### 填写指导

**Step 1 - 访问官网并截图**
使用浏览器打开产品官网，截图首页

**Step 2 - 提取品牌元素**
- Logo：描述图标形状、字体样式（不要猜测）
- 主色调：使用取色工具获取十六进制值
- 强调色：按钮、高亮等元素的颜色
- 整体风格：科技感、商务感、活泼感等

**Step 3 - 替换占位符**
将上述模板中的 [占位符] 替换为实际提取的内容

**Step 4 - 调整中心视觉**
根据产品核心功能，调整"Central visual"部分的描述

## 关键要点

1. **3D 产品可视化**：将产品核心功能转化为立体视觉元素
2. **品牌色贯穿始终**：背景、强调色、光效都要体现品牌色
3. **文字清晰可读**：使用 bold、shadow、高对比度确保中文清晰
4. **精致细节**：glass material、metallic finish、soft shadows 提升质感
5. **居中平衡构图**：主体居中，视觉重心稳定
