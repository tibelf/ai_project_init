# 策略 B：工作流程可视化

## 适用场景
笔记重点介绍"如何工作"、"数据流转"、"多步骤处理"时使用。

## 使用时机
- 笔记强调"工作原理"、"数据流转"、"多步骤"、"输入输出"
- 产品涉及数据转换、文档处理、AI 自动化
- 需要展示"从输入到输出"的完整流程

## 视觉结构

### LEFT（输入）→ CENTER（处理）→ RIGHT（输出）

**LEFT 部分**：展示多种输入源
**CENTER 部分**：展示处理过程
**RIGHT 部分**：展示输出结果

### 填写指导

**核心要点**：
1. 根据笔记内容识别输入、处理、输出各阶段的具体内容
2. LEFT：列出产品实际支持的输入格式
3. CENTER：选择最符合产品的处理视觉（形象的产品标识）
4. RIGHT：具体描述产品生成的输出形式
5. 所有占位符必须替换为从官网提取的真实品牌元素

## 示例场景对照

| 笔记中的关键词 | 配图中体现的元素 |
|--------------|----------------|
| "从多种格式提取数据" | LEFT: 展示 Excel、PDF、图片等多个输入图标 |
| "AI 自动处理" | CENTER: 展示发光的 AI 处理节点 |
| "生成结构化报告" | RIGHT: 展示整洁的表格/文档输出 |
| "一键完成" | 箭头流动感强，强调自动化 |

## 关键要点

1. **三段式布局清晰**："输入 → 处理 → 输出"一目了然
2. **金色箭头是灵魂**：必须清晰可见，强化流程感和数据流动
3. **每个阶段可辨认**：输入图标、处理器、输出结果都要具体明确
4. **适配流程描述**：根据笔记内容调整输入类型和输出形式
5. **强调自动化**：光芒、流动感营造智能处理氛围
