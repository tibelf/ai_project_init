# 图片 Prompt 模板库

本目录包含小红书配图生成的结构化 prompt 模板。

## 模板列表

### 策略 A：静态产品展示
**文件**：`strategy_a_static_product.md`

> ⚠️ 模板使用占位符示例，生成配图前必须访问官网提取真实品牌元素

**适用场景**：笔记强调产品界面、设计、最终效果

**使用时机**：
- 展示产品外观、UI、视觉设计
- 突出产品的设计美学和用户体验
- 强调功能列表、界面设计、最终效果

**特点**：3D 产品可视化 + 品牌色 + 精致细节

---

### 策略 B：工作流程可视化
**文件**：`strategy_b_workflow.md`

> ⚠️ 模板使用占位符示例，生成配图前必须访问官网提取真实品牌元素

**适用场景**：笔记强调工作原理、数据流转、多步骤处理

**使用时机**：
- 产品涉及数据转换、文档处理、AI 自动化
- 需要展示"从输入到输出"的完整流程
- 笔记提到"如何工作"、"数据流转"、"多步骤"

**特点**：LEFT → CENTER → RIGHT 三段式流程 + 金色箭头

---

## 如何选择

1. **阅读笔记内容**，识别核心卖点
2. **判断笔记重点**：
   - 强调"结果" → 使用策略 A
   - 强调"过程" → 使用策略 B
3. **读取对应模板文件**
4. **根据产品品牌元素填写模板参数**
5. **使用 `tools/generate_image.py` 生成图片**

## 使用流程

```bash
# 1. 选择策略后，读取模板
cat references/image_prompts/strategy_a_static_product.md
# 或
cat references/image_prompts/strategy_b_workflow.md

# 2. 根据模板填写 prompt

# 3. 生成图片
python3 tools/generate_image.py -p "完整的prompt" -o output.png
```

## 模板结构说明

每个模板文件包含：
1. **适用场景**：何时使用这个策略
2. **使用时机**：具体的判断标准
3. **Prompt 模板**：完整的结构化 prompt
4. **填写示例**：实际产品的完整示例
5. **关键要点**：注意事项和最佳实践
